## 📖 About

**Équipement Chefchaouni** is a clean and responsive front-end website built to showcase products and services of an equipment business in Chefchaouen, Morocco. The site is built with pure HTML, CSS, and JavaScript — no frameworks or backend required.

---

## ✨ Features

- 🎨 Modern and clean UI design
- 📱 Fully responsive layout (mobile & desktop)
- ⚡ Fast loading — pure HTML/CSS/JS, no dependencies
- 🖼️ Product/service showcase sections
- 📍 Located and tailored for Chefchaouen, Morocco

## 🚀 Getting Started

No setup required! Just clone and open in your browser.

1. **Clone the repository**
   ```bash
   git clone https://github.com/bahasalah255/equipement_chefchaouni.git
   ```

2. **Open the project**
   ```bash
   cd equipement_chefchaouni
   open index.html
   ```

> Or simply drag and drop `index.html` into your browser. ✅
